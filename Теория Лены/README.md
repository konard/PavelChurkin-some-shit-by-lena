# Теория Лены

Дискретная («сетевая») модель пространства: бесконечная трёхмерная решётка
одинаковых элементарных ячеек минимального объёма, переключающихся между
состояниями «внутри/снаружи». Соседние точки притягиваются, но не сливаются
свободно — слияние в общий объём разрешено только по правилу сумм кубов

$$x_1^3 + x_2^3 + \dots + x_k^3 = z^3,\qquad k \in \{3,\dots,8\}.$$

Цель — найти закономерности слияния и разделения ячеек на больших числах.

> 📌 По решению владельца **релятивистские (ОТО/СТО) допущения** репозитория
> удерживаются на этой стороне — Теории Лены, тогда как из ядра
> [Теории Виктора](../Теория%20Виктора/Ядро.md) они вынесены.

## Содержимое

* [`cube_merge_search.py`](./cube_merge_search.py) — расчётный скрипт: перебор
  представлений куба суммой $k$ кубов, поиск первого слияния для каждого $k$,
  подсчёт роста числа вариаций. Без внешних зависимостей.
* [`generate_visualization.py`](./generate_visualization.py) — генератор
  статической HTML-визуализации (использует данные из расчётного скрипта).
* `visualization.html` — самодостаточная статическая визуализация роста числа
  вариаций (открывается офлайн). Перегенерация:
  `python3 "Теория Лены/generate_visualization.py"`.
* [`fermat_iut_signatures.py`](./fermat_iut_signatures.py) — расчёты к отчёту об
  IUT и обобщённом уравнении Ферма: проверка 10 известных решений, трихотомия
  сигнатур, перебор-иллюстрация «конечной проверки». Без внешних зависимостей.
* `отчёт-<временная метка>.md` — отчёты с результатами:
  * [`отчёт-2026-06-14T17-27-26Z.md`](./отчёт-2026-06-14T17-27-26Z.md) —
    закономерности слияния и разделения ячеек (кубическая модель);
  * [`отчёт-2026-06-23T21-26-32Z.md`](./отчёт-2026-06-23T21-26-32Z.md) —
    применение IUT к обобщённому уравнению Ферма $x^r+y^s=z^t$ (issue #9).

## Ключевые результаты (кратко)

* Слияние двух точек невозможно (Великая теорема Ферма для куба) → минимум 3
  слагаемых.
* **Первое слияние во всей сети возможно только восьмёркой:**
  $8\cdot 1^3 = 2^3$.
* Число вариаций слияния растёт и с числом слагаемых $k$, и с масштабом $z$.
* Двухточечный случай $x^3+y^3=z^3$ — это сигнатура $(3,3,3)$ обобщённого
  уравнения Ферма $x^r+y^s=z^t$; для широкого (гиперболического) класса
  показателей IUT-оценки Мотидзуки сводят перечень к конечной проверке (244
  сигнатуры при $r,s,t\ge4$, 2446 в зоне гипотезы Била) — **условно**,
  поскольку IUT не признана консенсусом. Подробнее — в отчёте от 2026-06-23.

## Запуск

```bash
python3 "Теория Лены/cube_merge_search.py"
```
