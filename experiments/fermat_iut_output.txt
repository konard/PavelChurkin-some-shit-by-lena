=== 4. Связка с Теорией Лены (кубическая модель) ===
  Правило слияния Лены:  x_1^3 + ... + x_k^3 = z^3.
  Случай k=2  →  x^3 + y^3 = z^3  — сигнатура (3,3,3),
    1/3+1/3+1/3 = 1 → класс 'euclidean'.
    Решений нет (Великая теорема Ферма для куба) — это и есть
    «барьер двух точек» Теории Лены: слияние двойкой невозможно.
  Обобщённое уравнение Ферма x^r+y^s=z^t расширяет это правило на
  разные показатели; IUT даёт конечную проверку для гиперболической зоны.
