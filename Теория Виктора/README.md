# Теория Виктора

Раздел репозитория под наработки Виктора в рамках общей темы — построения
«теории всего» (эфирная теория, ОТО и СТО).

Ядро подхода: **эфир — непрерывный распределённый диэлектрик** с удельной
ёмкостью ε0 и удельной индуктивностью μ0; электрическое поле = поляризация этой
среды, магнетизм = «инерция» перекладывания поляризации при движении заряда.
По духу это очищенный от механики электромагнитный эфир Максвелла–Лоренца.

## Содержимое

* [`ether_checks.py`](./ether_checks.py) — расчётный скрипт: проверка числовых
  утверждений отчёта ($c=1/\sqrt{\varepsilon_0\mu_0}$, волновое сопротивление
  вакуума $Z_0=\sqrt{\mu_0/\varepsilon_0}$, красная граница фотоэффекта, длина
  волны де Бройля). Без внешних зависимостей.
* `отчёт-<временная метка>.md` — отчёты с разбором. Текущий:
  [`отчёт-2026-06-15T09-14-24Z.md`](./отчёт-2026-06-15T09-14-24Z.md) — анализ
  эфирной модели и её соотношение с силой Лоренца, правилом буравчика, силой
  Ампера, эффектом Холла, токами Фуко, поляризацией, эффектом Фарадея,
  сверхпроводимостью и эффектом Мейснера, фотоэффектом, спином и опытом
  Штерна–Герлаха, волнами де Бройля, лэмбовским сдвигом, EVO Шоулдерса и др.

## Ключевые результаты (кратко)

* Из образа «эфир = длинная линия (ε0, μ0)» **без механики** следуют две точные
  величины вакуума: $c=1/\sqrt{\varepsilon_0\mu_0}$ и
  $Z_0=\sqrt{\mu_0/\varepsilon_0}\approx 376{,}73$ Ом (проверено скриптом).
* Модель сильна как наглядный каркас **классической** электродинамики (знаки
  магнитных сил, эффект Холла) и совпадает по языку с КЭД-«поляризацией вакуума».
* Главные открытые вопросы: модель не порождает постоянную Планка $h$ (потому
  «квантовые» эффекты — фотоэффект, спин/Штерн–Герлах, спектры — вне её),
  требует явной лоренц-инвариантности и не включает слабое/сильное
  взаимодействия. Подробности и пометки 🟢/🟡/🔴 — в отчёте.

## Запуск

python3 "Теория Виктора/ether_checks.py"
