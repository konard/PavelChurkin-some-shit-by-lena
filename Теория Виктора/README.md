# Теория Виктора

Раздел репозитория под наработки Виктора в рамках общей темы — построения
«теории всего» (эфирная теория, ОТО и СТО).

Пока раздел пустой и ждёт наполнения. Предполагаемый формат материалов — такой
же, как у [Теории Лены](../Теория%20Лены/):

* длинные последовательные физические и математические расчёты;
* расчётные скрипты;
* отчёты в Markdown с временной меткой в названии;
* в перспективе — статические HTML-визуализации.

Когда появятся материалы, добавьте их сюда и обновите краткое резюме в
[корневом README](../README.md).
