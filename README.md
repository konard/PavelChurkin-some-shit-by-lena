# some-shit-by-lena

Репозиторий о разработке «теории всего» — эфирная теория вместе с ОТО и СТО.
Основная работа здесь — длинные последовательные физические и математические
расчёты; в перспективе к ним добавятся визуализации в виде статического HTML.

Материалы разнесены по двум разделам — по автору теории.

## Структура

### 📁 [Теория Виктора](./Теория%20Виктора/)

Эфирная электродинамика: эфир — непрерывный поляризуемый диэлектрик с удельной
ёмкостью ε0 и удельной индуктивностью μ0 (очищенный от механики эфир
Максвелла–Лоренца). Из образа «эфир = длинная линия» без механики следуют две
точные величины вакуума — $c=1/\sqrt{\varepsilon_0\mu_0}$ и волновое
сопротивление $Z_0=\sqrt{\mu_0/\varepsilon_0}\approx 376{,}73$ Ом. Первый отчёт
соотносит модель с силой Лоренца, эффектом Холла, токами Фуко,
сверхпроводимостью, фотоэффектом, спином, волнами де Бройля, EVO Шоулдерса и др.;
второй (по комментарию владельца) — с гравитацией как «эфирным градиентом»
(модель поляризуемого вакуума), попыткой вывести квант из размера электрона
(упирается в постоянную тонкой структуры $\alpha\approx1/137$) и идеями атома без
ядра / массы из «непроницаемости эфира». Всё честно помечено: где образ работает,
а где нужна квантовая/ядерная физика. Подробнее — в
[README раздела](./Теория%20Виктора/README.md).

### 📁 [Теория Лены](./Теория%20Лены/)

Дискретная модель пространства: бесконечная трёхмерная решётка элементарных
ячеек, которые сливаются в общий объём по правилу сумм кубов
$x_1^3+\dots+x_k^3=z^3$ ($k$ от 3 до 8). Цель — закономерности слияния и
разделения на больших числах.

Главное на сегодня: слияние двух точек невозможно (теорема Ферма для куба), а
самое первое слияние во всей сети возможно **только восьмёркой** —
$8\cdot 1^3 = 2^3$. Расчёты и подробный разбор — в
[README раздела](./Теория%20Лены/README.md) и в отчётах с временно́й меткой.
