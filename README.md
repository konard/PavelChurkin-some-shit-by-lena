# some-shit-by-lena

Репозиторий о разработке «теории всего» — эфирная теория вместе с ОТО и СТО.
Основная работа здесь — длинные последовательные физические и математические
расчёты; в перспективе к ним добавятся визуализации в виде статического HTML.

Материалы разнесены по двум разделам — по автору теории.

## Структура

### 📁 [Теория Виктора](./Теория%20Виктора/)

Раздел под наработки Виктора. Пока пустой, ждёт наполнения (расчёты, скрипты,
отчёты, визуализации). Подробнее — в [README раздела](./Теория%20Виктора/README.md).

### 📁 [Теория Лены](./Теория%20Лены/)

Дискретная модель пространства: бесконечная трёхмерная решётка элементарных
ячеек, которые сливаются в общий объём по правилу сумм кубов
$x_1^3+\dots+x_k^3=z^3$ ($k$ от 3 до 8). Цель — закономерности слияния и
разделения на больших числах.

Главное на сегодня: слияние двух точек невозможно (теорема Ферма для куба), а
самое первое слияние во всей сети возможно **только восьмёркой** —
$8\cdot 1^3 = 2^3$. Расчёты и подробный разбор — в
[README раздела](./Теория%20Лены/README.md) и в отчётах с временно́й меткой.
